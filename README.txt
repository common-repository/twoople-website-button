== Description ==

Twoople makes it easy for your website visitors to chat with you.  Add a simple live-chat widget to your website in seconds.  

* Enjoy simple, minimalistic functionality on a friendly, easy to use interface.

* Twoople works on all visitor devices.  Its responsive, HTML5 design fits any screen.

* Receive e-mail alerts and/or push notifications when visitors want to chat.

* Twoople's cloud-based dashboard allows you to manage chats without having to download software for your MAC or PC.

* Download the Twoople App for iPhone and Android when you're on the go.

* Use your Twoople PURL (personalized URL, eg.  twoople.com/johnsmith) to hyperlink images or text, on or off your website, to a chat window.

* FREE to use.


== Installation ==

== Installation ==
1. Visit [www.twoople.com](https://www.twoople.com) and create a Twoople address.
2. Install the Twoople plugin from WordPress directory and activate it.  
3. Enter your Twoople address and specify your widget's appearance.
